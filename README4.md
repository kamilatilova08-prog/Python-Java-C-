Лабораторная 4 Тема 1.4. Структуры данных. Работа в VS Code со структурами данных «деревья/графы»

Дерево как структура данных в программировании представляет собой иерархическую структуру, состоящую из узлов. Для представления дерева в языке Python используются классы, словари, списки, кортежи, именованные кортежи, классы данных, словари со списками, списки кортежей, словари словарей, JSON-структуры, Pandas DataFrame, массивы Numpy, списки объектов, цепочки атрибутов. Для представления дерева на языке C++ используются классы, структуры, шаблоны, контейнеры STL, умные указатели, структуры с указателями, std::vector, std::list, std::map, std::set, std::tuple, std::variant, битовые поля, объединённые структуры, указатели на функции, лямбда-выражения. Для представления дерева на языке JAVA используются классы, интерфейсы, универсальные классы, вложенные классы, классы-записи, ArrayList, LinkedList, HashMap, TreeMap, классы-перечисления, абстрактные классы, статические классы, анонимные классы, Stream API, необязательные цепочки.

Графы как структура данных в программировании — это нелинейная структура организации данных, состоящая из «вершин» и «рёбер» между ними. Для представления графа на языке Python используются классы, словари, списки, кортежи, матрицы, Pandas DataFrame, массивы Numpy, графы NetworkX, словари множеств, списки списков, JSON-структуры, словари кортежей, множества рёбер, списки рёбер, матрицы смежности. Для представления графа на языке C++ используются классы, структуры, шаблоны, контейнеры STL, умные указатели, std::vector, std::list, std::map, std::set, std::unordered_map, std::unordered_set, std::tuple, матрицы, битовые наборы, указатели на функции, Boost Graph Library, матрицы смежности. Для представления графа на языке JAVA используются классы, интерфейсы, универсальные классы, ArrayList, LinkedList, HashMap, TreeMap, HashSet, TreeSet, матричные классы, Stream API, необязательные цепочки, классы Enum, абстрактные классы, вложенные классы, классы Record, библиотека JGraphT.
