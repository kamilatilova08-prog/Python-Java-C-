Вариант 4. 0-1 Рюкзак с жадным алгоритмом
Задача: реализовать жадный приближенный алгоритм для 0-1 рюкзака. Отличие от дробного
в том, что предметы нельзя дробить.

____________
Определение
Жадный алгоритм для задачи 0‑1 «Рюкзак» — это приближённый алгоритм, который на каждом шаге выбирает предмет с наибольшей удельной стоимостью (отношение стоимости к весу),
если его можно поместить в рюкзак без превышения вместимости. Цель — максимально увеличить суммарную стоимость предметов при ограничении на общий вес.
________________
Объяснение работы алгоритма
1.	Алгоритм получает список предметов (с весом и стоимостью) и вместимость рюкзака.
2.	Алгоритм сортирует предметы по убыванию удельной стоимости (стоимость / вес).
3.	Алгоритм инициализирует переменные: общий вес (total_weight = 0), общая стоимость (total_value = 0), список выбранных предметов (selected = []).
4.	Алгоритм проходит по отсортированному списку предметов (цикл for idx, item in sorted_items:).
5.	На каждом шаге алгоритм проверяет, помещается ли предмет в рюкзак: total_weight + item['weight'] <= capacity.
6.	Если помещается, алгоритм:
o	добавляет индекс предмета в список выбранных (selected.append(idx));
o	увеличивает общий вес (total_weight += item['weight']);
o	увеличивает общую стоимость (total_value += item['value']).
7.	Алгоритм возвращает:
o	список индексов выбранных предметов;
o	общую стоимость;
o	коэффициент аппроксимации (0,5).
__________
Временная сложность
•	Сортировка предметов: : O(nlogn)  — доминирующая операция.
•	Проход по списку: O(n) — каждый предмет проверяется один раз.
•	Итоговая сложность: O(nlogn)+O(n)=O(nlogn) .

Почему именно O(n log n):

Сортировка предметов по удельной стоимости выполняется за O(n log n) - это самая медленная операция. Линейный проход по отсортированному списку занимает O(n), что быстрее.
По правилу доминирующих операций, общая сложность определяется самой медленной частью - сортировкой, поэтому итоговая сложность O(n log n).
